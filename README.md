# 🌐 Bruno López Abejón - Professional Portfolio & 360TourX Showcase

## 1. About this project
This repository contains the source code for my professional website. It serves as my digital headquarters, reflecting my current role as a Computer Engineering Student at UNIE and Co-Founder at 360TourX, specializing in Virtual Tours & Digital Experiences.

## 2. The problem it solves
A traditional resume or a standalone LinkedIn profile is often not enough to showcase visual and interactive tech products. This website solves the need for a centralized, interactive portfolio that allows clients, recruiters, and collaborators to directly experience my work and understand my value proposition.

## 3. Features & Functionality
The website acts as a digital storefront and personal brand hub. It:
- Clearly presents my professional positioning and focus.
- Showcases the services and interactive 360º virtual tours developed at 360TourX.
- Provides a direct and structured way to contact me for networking or business opportunities.

## 4. How to run it
This project is deployed live and is fully accessible from any device.
- **Live Version:** [https://brunolopezabejon.github.io/](https://brunolopezabejon.github.io/)
- **Local Execution:** To run this locally, you simply need to clone this repository (`git clone https://github.com/brunolopezabejon/brunolopezabejon.github.io.git`) and open the `index.html` file in any modern web browser. No complex server setup is required.

## 5. Technical Decisions
To build this digital architecture, I made several key decisions:
- **Deployment Strategy:** I chose GitHub Pages for hosting. This allows me to maintain continuous version control; every time I push a code update, the live website reflects the changes automatically.
- **Mobile-First Approach:** Consistent with my work at 360TourX, I prioritized a responsive design to ensure that the end-user experience is seamless on mobile devices, tablets, and desktops.
- **Clarity over Clutter:** I opted for a minimalist structure to avoid cognitive overload and guide the user's attention directly to the content.

## 6. What I learned
Developing and deploying this portfolio has been an excellent exercise in mental structuring and digital architecture. It has taught me how to align my technical skills (web development, Git version control, deployment) with my public narrative, ensuring that my GitHub, LinkedIn, and Web presence all reinforce the same professional message without contradictions.
